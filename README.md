## Publicar en GitHub Pages
1. En tu repositorio ve a **Settings → Pages**.  
2. En **Source**, selecciona:  
   - **Branch:** `main`  
   - **Folder:** `/ (root)`  
3. Guarda.  
4. Tu página estará disponible en la URL que indicamos arriba.
