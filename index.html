2. Qué hace este GPT

El “GPT Soporte Microsoft 365” tiene como objetivo:

  • Responder dudas y resolver problemas frecuentes relacionados con Microsoft 365 empresarial.
  • Usar como base documentos internos (PDF, TXT, etc.) y, cuando no existan instructivos internos, la información oficial de Microsoft y videos corporativos en YouTube publicados por la empresa.
  • Reducir al máximo las llamadas al área de soporte mediante guías paso a paso.

3. Datos personales que tratamos

Para brindar soporte tratamos datos mínimos proporcionados por el usuario dentro del chat:

  • Identificativos: nombre, apellidos, correo corporativo.
  • Soporte: breve descripción del requerimiento o incidencia.

De forma técnica, los sistemas involucrados pueden generar metadatos (fechas, registros de actividad, identificadores técnicos y auditoría) necesarios para operar los flujos y cumplir políticas corporativas.

No solicitamos contraseñas ni datos sensibles. Si el usuario los ingresa por error, le pedimos no enviarlos y procederemos a eliminarlos de los canales bajo nuestro control cuando sea posible.

4. Finalidades del tratamiento

  • Atender y resolver solicitudes sobre Microsoft 365.
  • Ejecutar acciones automatizadas cuando el caso lo requiera (p. ej., envío automático de correo a soporte).
  • Mejorar la calidad del servicio mediante análisis agregados y no identificables.
  • Cumplir obligaciones legales, de seguridad y auditoría.

5. Acciones y transferencias necesarias para prestar el servicio

Cuando el usuario lo solicita (p. ej., “recuperación de contraseña”, “problemas con Outlook/OneDrive”, etc.), el GPT puede activar flujos mediante Power Automate y Power Apps que:

  • Generan y envían automáticamente un correo a asis.sistemas@ducol.com.co con el formato corporativo:
    • Asunto: el correo corporativo del usuario.
    • Cuerpo: nombre, apellidos, correo corporativo y descripción del problema.
  • Usan conectores de Microsoft 365 (p. ej., Outlook/Exchange Online, SharePoint, Teams) para ejecutar acciones internas.

En estos casos, los fragmentos estrictamente necesarios del mensaje del usuario se envían a los servicios involucrados para completar la acción. Los proveedores principales son de Microsoft (Power Platform / Microsoft 365). Los videos compartidos son solo enlaces a contenido corporativo en YouTube.

6. Base jurídica

  • Interés legítimo de Ducol Group SAS en prestar soporte interno eficiente sobre Microsoft 365.
  • Consentimiento del usuario cuando solicita el envío de comunicaciones o la ejecución de acciones automatizadas.
  • Cumplimiento de obligaciones legales y de seguridad (auditoría, prevención y detección de incidentes).

7. Conservación

Conservamos los datos el tiempo estrictamente necesario para gestionar la solicitud y cumplir con obligaciones internas de auditoría y retención en Microsoft 365/Power Platform. Los correos y registros técnicos pueden conservarse conforme a políticas de TI y normativas aplicables.

8. Seguridad

Aplicamos controles técnicos y organizativos acordes a buenas prácticas de Microsoft 365 y Power Platform (p. ej., DLP, control de accesos, segregación de roles, registros de auditoría y resguardo de credenciales en entornos seguros). También promovemos el uso de MFA y revisiones periódicas.

9. Derechos de los titulares

Los colaboradores pueden ejercer sus derechos de acceso, rectificación, actualización, oposición o supresión a través de asis.sistemas@ducol.com.co. Atenderemos la solicitud conforme a la normativa aplicable y políticas internas.

10. Uso de servicios de terceros

  • OpenAI / ChatGPT: las entradas del usuario necesarias para una Acción pueden transferirse al servicio externo correspondiente. OpenAI no controla cómo esos terceros usan o almacenan los datos.
  • Microsoft (Power Platform, Microsoft 365): ejecución de flujos, mensajería y almacenamiento asociados al soporte.
  • YouTube: se comparten enlaces a videos corporativos; no se incrustan reproductores ni se recogen cookies desde esta página.

11. Niños y datos sensibles

Este servicio está dirigido a personal laboral adulto. No está diseñado para recabar datos de menores. Rogamos no enviar datos sensibles (salud, orientación sexual, creencias, números de tarjetas, contraseñas, etc.).
